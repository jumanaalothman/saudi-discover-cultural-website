# Discover Saudi Arabia – Interactive Cultural Website

## Overview

Discover Saudi Arabia is an Arabic interactive cultural website developed for the Internet Technologies course. The website introduces Saudi Arabia, its regions, culture, historical landmarks, and important places through a dynamic web interface.

The project includes public pages for visitors and an admin-side dashboard for managing website content. The website uses both client-side and server-side web technologies.

## Live Website

View the website here:

[Discover Saudi Arabia Website](https://saudi-discover.infinityfreeapp.com/index.php)

## Technologies Used

- HTML
- CSS
- JavaScript
- PHP
- MySQL
- Web Hosting
- InfinityFree

## Main Features

- Arabic website interface
- Home page introducing Saudi Arabia and the purpose of the website
- Regions gallery page with region cards
- Region/place details page
- JavaScript interactions such as night mode and filtering
- Dynamic content retrieval from the database
- Admin login page
- Admin dashboard for content management
- Add, update, and delete content features
- Session-based admin protection
- Confirmation messages after adding, updating, or deleting records
